# monday-community-archive

This is a static mirror of an archive of the original [community.monday.com](https://community.monday.com/).

The original community was replaced in April 2026 with a new version based on a different stack. The content was mainly deleted if not deemed relevant by the current community managers. There was however a large amount of good knowledge in there, so having a static version of this which is searchable is a useful resource.

- Generated by `scrape.py` & `scrape_categories.py` from Omer's database backed version of the archive
- Search index based on lunr.js, built by `build_search-index.js`

## Serve locally

```bash
sh ./serve.sh
```

then open http://localhost:8080/

## Hosted by Cloudflare

Using pages for the frontend, and r2 for `/api/*`
